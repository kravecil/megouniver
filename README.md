# megouniver

Парсер и анализатор данных о поступлении в российские вузы (ЛЭТИ, ГУАП, СПбПУ, КФУ, УУНИТ).

Скрипт собирает актуальные данные о конкурсных списках (количество мест, список абитуриентов с баллами и приоритетами) и анализирует шансы поступления на основе вашего кода.

## Возможности

- 📊 Автоматический сбор данных о конкурсных списках из официальных сайтов вузов
- 🎯 Анализ шансов поступления на каждую специальность
- 📈 Расширенная статистика по каждому вузу (свободные места, количество заявлений)
- 🏫 Поддержка 5 вузов: ЛЭТИ, ГУАП, СПбПУ, КФУ, УУНИТ

## Установка

### Linux / macOS

```bash
# Установите Python 3.14+ и uv (если еще не установлены)
# https://docs.astral.sh/uv/#installation

# Клонируйте репозиторий
git clone <репозиторий>
cd megouniver

# Установите зависимости
uv sync
```

### Windows + WSL2

1. **Установите WSL2 и Ubuntu** (если еще не установлены):
   - Откройте PowerShell от имени администратора:
     ```powershell
     wsl --install -d Ubuntu
     ```
   - Перезагрузите компьютер

2. **Установите Python и uv в WSL**:
   ```bash
   # В терминале Ubuntu/WSL
   sudo apt update && sudo apt upgrade -y
   sudo apt install -y python3 python3-pip curl
   
   # Установка uv
   curl -LsSf https://astral.sh/uv/install.sh | sh
   source "$HOME/.cargo/env"
   ```

3. **Клонируйте проект и установите зависимости**:
   ```bash
   cd /mnt/c/<путь_к_репозиторию>
   uv sync
   ```

## Настройка

Перед запуском укажите ваш уникальный код абитуриента в файле `config/settings.py`:

```python
STUDENT_CODE: int = <ваш_код>
```

## Запуск

### Основной анализ (рекомендуется)

```bash
# Запуск через uv
uv run python scripts/analyse.py

# Или через venv
.venv/bin/python scripts/analyse.py
```

Скрипт соберет данные со всех вузов и покажет:
- Статистику по каждому вузу
- Куда проходит конкретный студент

### Только статистика

```bash
uv run python scripts/stat.py
```

Скрипт покажет расширенную статистику по каждому вузу:
- Количество свободных мест
- Общее количество заявлений
- Количество заявлений с баллом выше заданного лимита

### Варианты запуска

- **Все вузы**: Закомментированные строки в `scripts/stat.py` можно раскомментировать
- **Отдельный вуз**: Укажите нужный вуз в списке `EDU_LIST`

## Требования

- **Python**: 3.14+
- **uv**: 0.11+ (рекомендуется для управления зависимостями)
- **Библиотеки**: aiohttp, beautifulsoup4, lxml, pandas

## Структура проекта

```
megouniver/
├── config/           # Конфигурация (student_code)
├── harvesting/       # Сбор данных (парсеры вузов)
│   └── edu/         # Конкретные вузы
├── analysing/       # Анализ данных
├── scripts/         # Скрипты запуска
│   ├── analyse.py   # Основной анализ
│   └── stat.py      # Статистика
├── tests/           # Тесты
└── pyproject.toml   # Зависимости
```

## Примечания

- Парсеры используют асинхронный подход с ограничением concurrent запросов
- Данные сохраняются в памяти (не кэшируются локально)
- Учитывается приоритет подачи заявлений и наличие льгот
